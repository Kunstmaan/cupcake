Bootstrap Modal

To prevent accidental closing of a Bootstrap Modal, for example while filling in a form, specify data-backdrop="static"

Example: data-backdrop="static"

Example: data-backdrop="true" or no data-backdrop attribute

The modal acts like the unmodified Bootstrap modal.

Example: data-backdrop="false"

The modal acts like the unmodified Bootstrap modal without a backdrop.
