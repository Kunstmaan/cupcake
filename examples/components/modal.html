Bootstrap Modal Strict Close

Extend Bootstrap Modal with a strict-close option to prevent closing the modal by accident while, for example, filling in a form.

Example: data-strict-close="true" or programmatically: $('#el').modal({ strictClose: true })

Can only be closed by clicking an element with: data-dismiss="modal". This typically is a 'close' button. ('x', 'Close')

Example: data-strict-close="false" or no data-strict-close attribute

The modal acts like the unmodified Bootstrap modal.
